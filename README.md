# 🌍 Traveling Rental Platform

A full-stack travel rental web application where users can explore, create, edit, and manage travel listings with secure authentication, image uploads, and reviews. This project is inspired by platforms like Airbnb and is built using Node.js, Express.js, MongoDB, and Cloudinary.

🔗 **Live Demo:** https://traveling-rental-platform-8x.vercel.app/

---

##  Preview

> Visit the live website to explore the application.

#  Features

-  User Authentication (Register, Login & Logout)
-  Create, Update & Delete Listings
-  Add and Manage Reviews
-  Cloudinary Image Upload
-  MongoDB Database Integration
-  Session-Based Authentication using Passport.js
-  Responsive UI
-  Express.js Backend
-  Input Validation using Joi
-  Flash Messages
-  Secure Routes & Authorization
-  RESTful Routing

# ⚙️ Installation

## Clone the Repository

```bash
git clone https://github.com/Ajay8x/Traveling-Rental-Platform.git
```

Go to the project folder

```bash
cd Traveling-Rental-Platform
```

Install dependencies

```bash
npm install
```

Create a `.env` file

```env
ATLASDB_URL=your_mongodb_connection_string

SECRET=your_session_secret

CLOUD_NAME=your_cloudinary_name
CLOUD_API_KEY=your_cloudinary_api_key
CLOUD_API_SECRET=your_cloudinary_api_secret
```

Start the server

```bash
node app.js
```

or

```bash
npm start
```
